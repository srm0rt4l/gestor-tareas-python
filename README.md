# GESTOR DE TAREAS EN PYTHON

Este es mi primer proyecto con python, ha sido principalmente creado para tener un primer contacto con este lenguaje de programación y sobretodo con GitHub.


## ¿QÚE HACE ESTE PROGRAMA?

Es un programa muy simple que sirve para organizar tus tareas diarias. Puedes;

- ✅ Ver todas tus tareas
- ➕ Añadir nuevas tareas
- ✔️ Marcar tareas como completadas
- ❌ Eliminar tareas
